TIPS
----------------------------------------------------------
- Keep the same file names exactly (small letters, .jpg).
- If your file is .png, either rename it to .jpg, or tell
  your developer to update the file name in the code.
- Keep each photo under ~500 KB so the site loads fast
  (you can compress at tinypng.com for free).
